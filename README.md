### Create A New Django Project and APP
安装Django依赖

    $ pip install Django==2.0.8
创建一个Django项目
    
    $ django-admin startproject mysite
运行测试服务器

    $ python manage.py runserver
创建一个APP

    $ python manage.py startapp polls
